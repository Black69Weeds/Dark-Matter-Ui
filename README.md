# DarkMatter UI Library

DarkMatter is a modern, dark-themed, mobile-friendly UI library for Roblox scripts and executors.

- ✅ Draggable window  
- ✅ Tabs with two columns (Left / Right)  
- ✅ Sections to organize settings  
- ✅ Toggles, Sliders, Dropdowns, Textboxes, Labels  
- ✅ Works on PC + Mobile
